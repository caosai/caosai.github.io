java中开发者如何排序

发表于 2015-06-27   |   分类于 Simple Java   |  

分析开源java项目的大量开源代码，我发现java开发者常用的排序方法有两种。一种是用Collections或者Arrays的sort()方法，另一种是用排好序的数据结构，例如TreeMap和TreeSet。

使用sort()方法

集合使用Collections.sort()方法

数组使用Arrays.sort()方法

当集合或数据创建好了，这种方式非常简洁。

使用排序好的数据结构

list或者set使用TreeSet排序

map使用TreeMap排序,TreeMap 根据key排序

这种方法是非常有用的,如果你想做很多关于集合的搜索操作。排序的数据结构的时间是复杂度O(logn),低于O(n)。
